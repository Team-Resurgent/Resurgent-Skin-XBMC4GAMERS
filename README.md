## System Requirements
### Minimum
* XBMC4GAMERS.

## Install Notes
* [Download X4G-Resurgent](https://github.com/Team-Resurgent/Resurgent-Skin-XBMC4GAMERS/releases)
* Unzip Content
* Place unzipped folder into "XBMC4Gamers/skins/profile skin/"
* Win!!!
